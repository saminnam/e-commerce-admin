This project is a user management portal built with React.js, Node.js, Express, MongoDB, and Axios.

I have implemented login authentication where the admin can log in by default using a predefined username and password. The admin can add multiple users, each with their own username and password. These users can then log in to the portal with their credentials.

This setup provides a full-stack authentication flow along with user management functionality in a responsive React-based interface.
